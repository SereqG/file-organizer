# Data Pipeline Project

A modular data ingestion and processing pipeline.

## Getting Started

```bash
pip install -r requirements.txt
python main.py
```

## Configuration

Edit `config.json` to set database connection, log level, and pipeline parameters.

## Project Structure

- `main.py` — application entry point
- `utils.py` — shared helper functions
- `index.ts` — TypeScript API server
- `helpers.tsx` — React UI components
